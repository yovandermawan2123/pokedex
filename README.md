# Pokedex App – Flutter

A simple Flutter application built using the PokéAPI.  
This app displays a list of Pokémon along with detailed information such as types, stats, abilities, and moves.  
It serves as a demonstration of API integration, state management, and modern Flutter UI.

---

## 🔧 How to Run the Project

1. Open the project folder in your terminal  
2. Install dependencies:
```
flutter pub get
```
3. Run the app:
```
flutter run
```

---

## 🛠 Build APK

```
flutter build apk --release
```

The APK will be generated at:
```
build/app/outputs/flutter-apk/app-release.apk
```

---

## 🌐 API
Using PokéAPI:  
https://pokeapi.co/

---

## ✨ Features
- Pokémon list display  
- Detailed Pokémon information  
- Stats, Moves, Abilities  
- Clean and modern UI
